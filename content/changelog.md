_History of updates to the Vector Charts cloud service._

## April 2026

Lighthouse light sectors and public chart updates.

- Lighthouse light sectors render on the map; toggle them in the style editor
- Public chart updates are now ingested once per week

## March 2026

Public chart updates and style customization.

- Customize sounding colors in the style editor
- Public chart updates appear automatically when new data is available

## February 2026

Added custom style editor and account settings.

- Customize chart appearance with your own colors, icon sizes, opacity, and imported styles (Business plan)
- Choose which email notifications you receive in account settings

## December 2025

More chart sources and style control.

- Added US inland waterway charts with expanded symbols and basemap detail
- Added API parameter to show or hide specific chart layers via the style API
- Updated getting-started guides in the portal

## November 2025

Pricing and Business plan APIs.

- Updated usage-based pricing.
- Query, GeoJSON, shoreline, and routing APIs available on the Business plan

## October 2025

Improved chart rendering, and metadata API.

- Added attributions endpoint for chart data sources
- More chart symbols and clearer icons, areas, and depth contours
- Portal UI: Added a Terms of Service agreement when signing in
- Portal UI: View pending invoices in the portal

## September 2025

Tweaks to chart rendering and billing.

- Render buoy marks and other symbols render more accurately on the map
- Increased API usage limits by plan

## August 2025

Added Business account APIs for raw data manipulation.

- Added GeoJSON query, elevation query, and general query API endpoints

## July 2025

Added Team / Business accounts, improvements to chart licensing.

- Share an account with your team; billing and usage are tracked at the account level
- View invoice history and monthly usage in the portal
- Added a feature request website
- Added internationalization/units: show depth labels in meters, feet, or fathoms
- Improved symbology of unsafe areas on the map
- Improved basemap coverage at lower zoom levels; map now covers the entire world
- Browse available charts in the portal

## April 2025

Added more granular API token security.

- Added feature to restrict API tokens to specific origins

## March 2025

Pricing and basemap.

- Updated the usage-based subscription pricing.
- Added symbology for Airports.

## February 2025

Improved rendering.

- Tweaked depth sounding font; soundings display more clearly
- Increased the maximum rendering of tiles to Z 14.
- Added overscale rendering to display tiles above Z 14.

## September 2024

Initial launch of Vector Charts Cloud!

- Added web portal: sign up, create API tokens, and track usage from the portal
- Added API server: Vector and raster chart tiles available via the API
- Initial release of documentation and setup guide
- Added a free tier to the API so you can try Vector Charts before subscribing
- Added a graph to track API usage in the portal UI
- Added warning when API usage is approaching your account's limit
