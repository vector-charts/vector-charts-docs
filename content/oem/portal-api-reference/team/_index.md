The Team endpoints manage the users on a Vector Charts OEM account. Use them to invite new members, list the current team, and rotate your own password.

Inviting a new team member sends an onboarding email to the invitee with a link to set their password. Inviting requires a working SMTP configuration on the OEM instance.
